## 🎓 Academic Coding Exercises - Estácio University

***📖 About***

This repository contains a collection of exercises and projects developed during my first semester of the Computer Science/IT degree at Estácio University. It serves as a record of my foundation in logic and initial software development.


***📂 Repository Structure***

The files cover fundamental programming concepts:

**Web Development:** Basic layouts using HTML5 and interactivity with JavaScript (e.g., Atividade_JS.html).

**Python Logic:** Decision-making structures, loops, and data handling (e.g., Par_e_Impar.py).

**GUI Basics:** Introduction to graphical user interfaces using the Tkinter library.

**Automation:** Basic scripts for email handling and web requests.


***🛠️ Tech Stack***

**Frontend:** HTML5, CSS, JavaScript.

**Scripting:** Python 3.x.

***📌 Archival Note***

This is an academic repository and is currently archived. It represents the beginning of my technical journey and is kept here for portfolio purposes.
